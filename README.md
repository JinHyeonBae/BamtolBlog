# BotLineBlog

## Description
일기, 공부, 하루의 일상을 적어올리는 블로그 홈페이지를 제작하는 목적으로 제작되었습니다.

- Front : React
- Backend : Spring Boot


## Installation

### Frontend

1. 폴더 이동
```c
cd front
```

2. 모듈 설치
```c
npm install
```

3. 실행
```c
npm start
```

### Backend

1. 폴더 이동
```c
cd backend/spring
```

2. 실행
```c
./gradlew bootrun
```
